# 💈Barbearia
## Sobre o projeto
Esse projeto faz parte de uma série em que recupero designs antigos, melhoro eles e os desenvolvo com ferramentas atuais, buscando aprimorar e demostrar minhas habilidades.
Também estarei disponibilizando o Figma do projeto!

## 🚀Deploy
Deploy pela vercel, pode acessar no link abaixo:
